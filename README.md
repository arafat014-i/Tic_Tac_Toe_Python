🎮 Tic Tac Toe Game
A simple Python game built using Tkinter where two players can play Tic Tac Toe in a graphical interface.

Features
2-player game
GUI using Tkinter
Winner detection
Simple and interactive interface

Installation
git clone https://github.com/arafat014-i/tic-tac-toe-python.git
cd tic-tac-toe-python
python tic_tac_toe.py
